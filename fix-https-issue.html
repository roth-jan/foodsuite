FoodSuite AWS - Korrektur für HTTPS Problem

Bitte verwende diesen Link:

http://18.195.206.72:3005

Alternativ - Quick Fix:

Falls du bereits auf der Seite bist, führe diesen Code in der Browser-Konsole aus:

// Fix API URLs to use HTTP instead of HTTPS
window.API_BASE_URL = 'http://18.195.206.72:3005/api';
window.apiClient.baseURL = 'http://18.195.206.72:3005/api';

// Reload the current tab
if (window.currentTab) {
    window.showTab(window.currentTab);
}

Permanente Lösung:

Die foodsuite-complete-app.html muss angepasst werden:

// Zeile 9791 ändern von:
const API_BASE_URL = window.location.origin + '/api';

// Zu:
const API_BASE_URL = 'http://18.195.206.72:3005/api';
